# College_Chatbot

🎓 College Chatbot using Python & Flask

📌 Project Overview

The College Chatbot is an intelligent web-based chatbot designed to answer frequently asked questions related to a college. It provides instant responses to users by retrieving information from a predefined knowledge base, making it easier for students and visitors to access important information.
This project demonstrates how Python and Flask can be used to build a simple, interactive chatbot for educational institutions.

🚀 Features

💬 Interactive chatbot interface

🎓 Answers college-related queries instantly

📚 FAQ-based response system

🗄️ SQLite database integration

🌐 User-friendly web interface

⚡ Fast and lightweight application

⚙️ Project Workflow

User enters a question.

Flask receives the request.

The chatbot searches the FAQ database.

The most relevant answer is retrieved.

The response is displayed on the web page.

▶️ How to Run the Project

Clone the repository.

Install the required dependencies:

pip install -r requirements.txt

Run the application:

python desk.py

Open the application in your browser.
